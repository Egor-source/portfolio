OdmioJs - это высокоуровневая, строго типизированная ODM (Object-Document Mapper) библиотека для MongoDB, разработанная на TypeScript. Проект построен вокруг архитектурных принципов SOLID, использует паттерн Data Mapper и предлагает полную независимость бизнес-логики от конкретной реализации базы данных благодаря абстракции драйверов.

##  Ключевые особенности

* **Monorepo Architecture:** Архитектура монорепозитория на базе Yarn 4 (Berry). Код разделен на изолированные слабосвязанные пакеты (`core`, `types`, `drivers`), что упрощает масштабирование и публикацию.
* **Driver Agnostic (Инверсия зависимостей):** Бизнес-логика и репозитории работают исключительно через интерфейсы. Вы можете бесшовно переключаться между реальным `MongoDriver` и `InMemoryMongoDriver` (для изолированного тестирования без поднятия Docker/реальной БД).
* **Data Mapper Pattern:** Разделение ответственности - сущности приложения не знают, как они сохраняются в базу данных. Вся работа с данными инкапсулирована в репозиториях.
* **Type-Safe Queries:** Максимальное использование возможностей TypeScript для обеспечения строгой типизации схем, документов и поисковых запросов.

##  Структура проекта

Проект организован в виде монорепозитория со следующей структурой пакетов:

* `packages/core` - основное ядро библиотеки: логика клиента, базовые репозитории и оркестрация.
* `packages/types` - общие интерфейсы, типы данных и контракты для драйверов и ядра.
* `packages/drivers` - реализации адаптеров баз данных (включая рабочий драйвер для MongoDB и In-Memory решение).

##  Архитектурный подход

В отличие от классического паттерна Active Record (используемого, например, в Mongoose), где модель содержит и данные, и методы работы с базой, OdmioJs реализует паттерн **Data Mapper**:

1.  **Интерфейсы драйверов:** Ядро (`core`) определяет контракт, которому должен соответствовать любой драйвер.
2.  **Репозитории:** Доступ к коллекциям осуществляется через `Repository`, что изолирует слой персистентности.
3.  **Тестируемость:** Благодаря `InMemoryMongoDriver` интеграционные тесты выполняются мгновенно в оперативной памяти и не оставляют побочных эффектов.

## ️ Быстрый старт (Пример использования)

```typescript
import { OdmioClient } from '@odmiojs/core';
import { MongoDriver } from '@odmiojs/mongo-driver';

// 1. Инициализация клиента с нужным драйвером
const client = new OdmioClient({
  driver: new MongoDriver({ uri: 'mongodb://localhost:27017/mydb' })
});

await client.connect();

// 2. Получение репозитория для коллекции
interface User {
  id: string;
  name: string;
  email: string;
}

const userRepository = client.getRepository<User>('users');

// 3. Работа с данными
const newUser = await userRepository.create({
  name: 'Egor',
  email: 'egor@example.com'
});

const user = await userRepository.findOne({ name: 'Egor' });
console.log(user);

await client.disconnect();
```
