**Smail Shop** is a large-scale e-commerce platform built for a leading manufacturer of children's technical apparel (membrane, fleece, knitwear) and accessories. The platform concurrently serves B2B and B2C segments, delivering entirely distinct interface environments and business logic based on the user’s authentication state.

### My Role & Key Contributions (Frontend)

As a Frontend Developer, I was responsible for engineering the dynamic catalog interface, building interactive customer tools, optimizing full-text search components, and developing a feature-rich B2B partner dashboard.

#### Key Features & Implemented Modules:

1. **Dual-State Dynamic Catalog UI (B2C / B2B):**
   - **B2C (Pre-authorization):** High-impact visual grid with large product cards and rich media layouts focusing on lifestyle presentation, restricting checkouts to the product detail page.
   - **B2B (Post-authorization):** A high-load _"Quick Order Sheet"_ interface utilizing a compact list view and structural sizing matrices for rapid, multi-item wholesale purchasing.

2. **"Dressing for Bad Weather" Interactive Guide:**
   - Developed an interactive widget on the homepage that provides gamified value to parents while acting as a highly optimized, direct entry point into targeted product categories.

3. **Smart Navigation & Mega-Menu:**
   - Engineered an advanced navigation structure divided by categories and age brackets, automatically displaying target size sets for each group.
   - Embedded informative content banners into menu layers to showcase fabric technologies (e.g., _Softshell_ breakdown).

4. **Advanced B2B Customer Portal (Dashboard):**
   - **3-Click Checkout:** Streamlined order placement flows equipped with automatic user profile data-population and minimum order value constraints.
   - **Tiered Pricing Matrix:** Dynamic front-end logic representing multiple price tiers. Integrated a visual incentive tracker displaying the remaining order value needed to unlock the next discount level.
   - **Batch Actions:** Tabbed navigation between _Favorites_, _Popular items_, and the catalog matrix, allowing users to move multiple variations to the cart in a single click.
   - **Collapsible Promo Feed:** An interactive accordion-based banner feed for active corporate campaigns.

5. **Order Lifecycle & Document Management:**
   - Designed dense data grids for order history with deep filtering (by ID, date, status), order cancellation, and replication routines.
   - Integrated on-demand generation and downloading of financial document packages (Invoices, UPD, receipts).

6. **Search & Content Control Systems:**
   - Wired frontend components into a real-time full-text **Elasticsearch** engine to handle complex, nested JSON-formatted product data smoothly.
   - Built horizontal overlay banners driven by a strict client-side prioritization matrix (Tiers 1, 2, 3) to control the marketing narrative.

7. **Social Mechanics & Forms:**
   - Created streamlined 2-field rapid checkout forms and an interactive product evaluation module supporting user-uploaded rich media (photos/videos).
