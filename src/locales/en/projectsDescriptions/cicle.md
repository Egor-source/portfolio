A secured multi-role B2B e-commerce platform built to automate wholesale purchasing cycles for garden and household plastic hardware. The system establishes a direct, algorithmic commercial link between large retail chains, dealers, and the corporate 1С:ERP backend, managing custom pricing matrices, operational draft workflows, and structured shipment pipelines.

### My Role & Key Contributions

As a Frontend Developer, I engineered the client-side interaction architecture for complex wholesale logic, built reactive components for freight-load planning, structured heavy data-table views, and developed an advanced multi-profile dashboard separating 7 distinctive access roles.

### Key Implemented Modules & Features

* **Reactive Transport Load Indicator:** Programmed a live tracking widget that visualizes the current volumetric capacity percentage of a freight vehicle as items are appended to the cart, incorporating multi-state color codes and alert messages for cargo limit overruns.
* **High-Density Professional Catalog Grid:** Developed an alternative spreadsheet-style layout for bulk stock selection, highly optimized for manual keyboard entries, featuring client-side persistence of the preferred viewing mode inside cookies.
* **Multi-Stage Order Draft Management:** Engineered a reactive draft interface allowing users to create infinite separate carts, track asynchronous price/stock validation states linked to the 1C:ERP node, and trigger 1-click conversions into live corporate orders.
* **Consolidated Shipment Builder:** Created a consolidation wizard interface that merges diverse bulk orders across multiple corporate sub-entities into a single freight machine entity, dynamically compiling associated split-payment timelines.
* **Adaptive Search Filters & Labels:** Integrated an advanced parametric filter system that recalculates matching inventory counts on the fly, paired with corporate localization filters that flag items previously bought or tailored for specific accounts.
* **Accounts Receivable & Audit Tracker:** Designed an interactive financial settlement zone displaying corporate debt limits, calendar milestones for upcoming payments, and system-wide UI warning banners that freeze ordering triggers if hardcopy legal documents are overdue.
* **Multi-Role Analytics Dashboard:** Structured variable landing layouts adapted to active access authorization tiers, equipped with retro-bonus milestone tracks, cross-company asset allocation pie charts, and global top-bar marquee announcement banners.
