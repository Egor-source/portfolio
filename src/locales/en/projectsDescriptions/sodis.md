**Sodis-Camp** is a high-end e-commerce travel platform tailored for discovering and booking bespoke tours, worldwide expeditions, fishing, and hunting trips. Designed for premium audiences, the platform requires an immersive user experience, heavy multimedia optimization, seamless map integrations, and highly custom catalog interaction mechanics.

### My Role & Key Contributions (Frontend)

As a Frontend Developer, I focused on turning design concepts into high-performance client-side logic, managing rich data layouts, and developing interactive map-driven catalog elements.

### Key Implemented Modules & Features:

- **Interactive Dynamic Preview Cards:**
  - Engineered custom product cards for the tour catalog that enable users to preview map routes directly within the list.
  - Implemented an inline swipeable photo gallery inside each preview card, allowing users to browse photos without opening the detailed page.

- **Geospatial & Mapping Implementations:**
  - Integrated Yandex.Maps to power an interactive, country-based discovery map filter on the homepage.
  - Developed dedicated route maps on detailed tour and hotel pages, equipped with specialized milestone markers representing specific tour activities.

- **Rich Media & Video Layouts:**
  - Built user interaction logic for high-definition atmospheric background video headers linked to specific catalog items.
  - Developed special-offer carousels and independent image scrollers nested within promo card modules.

- **Advanced Tiered Filtering & Tagging:**
  - Developed efficient client-side filtering systems for three primary vacation sectors (travel, fishing, hunting).
  - Processed multi-variable inputs for rapid data filtering (season, visa requirements, duration, country).
  - Wired tag-based quick navigation controls to swap between category presets on the homepage.

- **Contextual Widgets & Forms:**
  - Implemented real-time regional weather widgets on detailed tour pages.
  - Structured day-by-day itinerary grids and multi-field booking forms for tours and hotels.

- **Performance-Driven Responsive Layout:**
  - Optimized heavy-asset web layers (simultaneous video loops, custom maps, and high-res imagery) to guarantee smooth transitions and fast responses.
  - Maintained perfect consistency and adaptation across mobile and desktop viewpoints (Pixel Perfect alignment).
