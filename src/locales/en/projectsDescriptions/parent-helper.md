**Parent Assistant** is a comprehensive, public-facing digital platform and verified knowledge base designed to guide parents through children's educational and talent development pathways. The platform provides tools for digital aptitude diagnostics, extracurricular matching, and instant access to over 10,000 learning materials and live national student olympiad calendars.

### My Role & Key Contributions (Frontend)

As a Frontend Developer, I engineered high-density data views, dynamic search directories, and complex reactive components utilizing the Next.js framework to deliver an optimized user experience.

#### Key Implemented Modules & Features:

- **Digital Library & EER Directory:**
  - Built the user interface for a large-scale catalog containing verified learning materials.
  - Implemented multi-variable query selectors (by Russian regions, grades, subjects, and resource types) with real-time UI state updates.

- **Interactive Competition & Olympiad Calendar:**
  - Developed an advanced event tracking engine featuring a horizontal daily timeline swiper.
  - Implemented pre-set time range toggles (this week, month, next week) and cross-category filtering layouts.

- **Student Aptitude & Diagnostic Testing UI:**
  - Designed and developed form wizard flows and survey components to handle digital diagnostics aimed at identifying children's talents and skills.

- **Responsive Data Cards & Carousels:**
  - Created modular grid card components tracking view counts, user ratings, and favoriting triggers.
  - Developed touch-responsive carousels for recommended reading items and promotional banners.

- **Next.js Performance Fine-Tuning:**
  - Handled optimization strategies for heavy content sections and list rendering.
  - Leveraged native asset optimization and efficient handling of collapsible accordions (FAQ section) and heavy dropdown arrays.

- **Mobile-First Responsive Optimization:**
  - Achieved strict mobile device compliance, ensuring dense tabular configurations and filter drawers remain fully legible and accessible on narrow viewports.

### Core Platform Capabilities

- **Aptitude Diagnostic Engine:** Interactive test models that assess a child’s profile to suggest personalized extracurricular programs.
- **Verified E-Learning Repository:** A structured archive for courses, instructional articles on arts and sciences, and self-study manuals.
- **National Event Tracker:** System-wide registry of state and regional academic competitions classified by level of difficulty and student age groups.
