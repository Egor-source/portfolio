A public-facing web portal for an educational institution designed to manage parent-student onboarding, showcase academic achievements, and maintain mandatory state compliance records. The platform integrates structural information directories, automated admission forms, media galleries, and a comprehensive digital accessibility layer.

### My Role & Key Contributions

As a Frontend Developer, I handled UI/UX styling according to brand guidelines, developed custom interactive components, engineered client-side form logic, and maintained cross-browser web accessibility standards.

### Key Implemented Modules & Features

* **Accessibility Mode for Visually Impaired:** Built a dedicated assistive layout complying with state GOST and WCAG 2.1 accessibility benchmarks. Implemented adjustable contrast profiles, dynamic font scaling controls, and image-hiding triggers to aid content readability.
* **Admission & Scholarship Form Wizards:** Developed client-side web forms for submitting prospective student enrollment applications, financial scholarship requests, and job applicant resume uploads with built-in form field validations.
* **Faculty Directory & Bio Cards:** Created an interactive team directory with dynamic sorting for school administration and academic staff, linked with independent detailed biography views containing interactive contact routes.
* **Live Search with Instant Suggestions:** Implemented a UI component for real-time, global text search that renders interactive search outputs within an overlay layer without full-page reloads.
* **Dynamic Statistics Grids:** Built responsive data tables showcasing student performance scores on state graduation exams (OGE/EGE) over consecutive fiscal academic years.
* **News Dashboard & Event Galleries:** Engineered an automated press feed filtered by custom news taxonomies, coupled with photo albums indexed by event dates and interactive article discussion components.
* **Location Mapping & Core Routing:** Handled Yandex.Maps API integration for the branch locator view, implementing custom multi-modal route calculations from the nearest transit hubs and styling external links leading to the private student dashboard.

### Core Platform Capabilities

The user interface fulfills strict regulatory architecture parameters required for government-inspected educational resources. Global navigation maps out easy pathways to legal compliance documents, while the underlying grid layout follows strict mobile-responsive frameworks to display calendar configurations and image feeds legibly on smaller viewports.
