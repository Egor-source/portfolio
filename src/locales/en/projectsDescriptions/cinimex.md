The official corporate web portal for an enterprise-grade IT software vendor and systems integrator. The platform is designed to showcase custom software architecture solutions, analyze completed case studies, handle B2B lead generation, and drive human resource acquisition pipelines. The system merges high-density service directories, an online press room, and a scalable recruitment environment.

### My Role & Key Contributions

As a Frontend Developer, I engineered the interactive user interface layers, developed the modular block-driven page constructor layout, wired cross-entity data relations, and integrated advanced geospatial tracking utilities.

### Key Implemented Modules & Features

* **Block-Based Layout Constructor:** Engineered a flexible suite of standalone UI blocks, allowing marketing teams to dynamically assemble, arrange, and style product solution pages and project case studies differently using pre-built templates.
* **Cross-Entity Catalog Linking:** Implemented comprehensive front-end relational schemas to link core enterprise software solutions with specific client success logs, active press releases, and corporate media mentions.
* **Geospatial Route Mapping Component:** Handled Google Maps API implementations to display an interactive corporate branch directory. Programmed location-based filters and a reactive utility that calculates and renders pedestrian path coordinates leading from the nearest subway stations to the office entrance.
* **Full-Feature Localization Mirroring:** Designed the architecture supporting a flawless, zero-friction English version of the portal that completely mirrors the complex cross-linking structures, interactive maps, and forms found in the Russian release.
* **Multi-Category Press Room:** Created a responsive content hub equipped with runtime category filters (news, public articles, corporate events) alongside modular scripts that map specific announcements onto related IT software profiles.
* **HR Career Hub & Social Feeds:** Developed a specialized career acquisition zone combining employee success stories, grade-filtered job openings, resume attachment upload dialogs, and a custom media integration utility tracking external photo feeds.
* **Operational B2B Contact Forms:** Installed targeted validation forms for rapid project consultations, custom feedback fields, and localized Telegram support bridge routing controls for both global markets.
